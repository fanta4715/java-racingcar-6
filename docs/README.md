# 기능 구현 목록

## CarView (View) 
의존성 x
- [ ] 이름 입력 받는 문장 출력
- [ ] 시도할 회수 받는 문장 출력
- [ ] 실행 결과 출력 (컨트롤러로부터 데이터 받음)
- [ ] 최종 우승자 출력

## Car (Model)
의존성 x
- [ ] 이름, 이동 횟수 저장
- [ ] 랜덤 값 생성 기능
- [ ] 값이 4보다 클 경우 이동 횟수를 +1하는 기능

## CarController (Controller)
Model과 View에 의존성을 가짐
- [ ] 자동차 리스트에 저장
- [ ] 시도 횟수 만큼 실행 결과 출력하기
- [ ] start() 함수 구현
- [ ] 사용자로부터 입력된 값의 유효성 검증
